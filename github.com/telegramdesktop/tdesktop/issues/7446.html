New message notifications appear even when app in focus, potentially hiding app itself #7446

amaau opened this issue Mar 25, 2020 · 0 comments

Comments

Copy link

amaau commented Mar 25, 2020

Steps to reproduce

  1. Have the telegram window docked on the right side of the screen
  2. Be in a chat with someone
  3. Have someone else write you a new message

Expected behaviour

No notification appears, since you're already focused on Telegram and can see the new message indicator in the chats list

Actual behaviour

New message notification appears. If your app is on the right side of the screen, the notification bubble hides the bottom part, that is - the place where you're currently typing, forcing you to hide / hide all, in order to continue.

Configuration

Operating system:
Windows 10 64

Version of Telegram Desktop:
1.9.2.1

Used theme:
Dark
